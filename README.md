# tlbb
天龙八部游戏源码

## 项目简介
天龙八部游戏源码是07年比较火的一部仙侠类网游，由搜狐游戏公司开发。由于该源码是泄露出来的，并非官网开源，所以上传源码可能会造成非法传播等侵权问题，故个人只是作为学习使用，所有仓库就不上传该游戏源码了。由于该游戏开发年代比较久远了，故使用现代编译器进行编译会产生许多的兼容性问题，目前已经被我修复完成了，修复记录可以查阅**天龙八部编译记录.md**
